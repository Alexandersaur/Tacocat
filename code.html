                            //--Get user string from the page (controller function)--//
                            function getValue(){
                                //--Make sure the alert is invisible--//
                               document.getElementById("alert").classList.add("invisible");
                               let userString = document.getElementById("userString").value;
                               //--Check for a palindrome--//
                               let returnObj = checkPalindrome(userString);
                               //--Display the results to the screen--//
                               displayMessage(returnObj);
                           }
                           
                           //--Reverse the string and check for palindrome (logic function)--//
                           function checkPalindrome(userString){
                               //--Ensure only alphanumeric characters are compared, no spaces or symbols--//
                               userString = userString.toLowerCase();
                               let regex = /[^a-z0-9]/gi;
                               userString = userString.replace(regex,"");
                               //--Reverse the cleaned string using a for loop--//
                               let revString = [];
                               let returnObj = {};
                               for (let index = userString.length - 1; index >= 0; index--) {
                                   revString += userString[index];
                               }
                               //--Determine if the string is a palindrome and update the object msg property--//
                               if(revString == userString){
                                   returnObj.msg = "SUCCESS! You entered a palindrome!"
                               }else{
                                   returnObj.msg = "Sorry, you did not enter a palindrome."
                               }
                               returnObj.reversed = revString;
                               return returnObj;
                           }
                           
                           //--Show the result to the user (display function)--//
                           function displayMessage(returnObj){
                               //--Write to page--//
                               document.getElementById("alertHeader").innerHTML = returnObj.msg;
                               document.getElementById("msg").innerHTML = `Your phrase in reverse is: ${returnObj.reversed}`;
                               //--Show the alert box--//
                               document.getElementById("alert").classList.remove("invisible");
                           }
getValue()

This is a function that was created to collect the string information from the user via the input element contained within the form. It will display the final result after calling the other two functions to perform their actions.

checkPalindrome()

This is a function that was designed to create an Object containing 'reversed' and 'message' properties. First it cleans the user input by making everything lower case and removing spaces and special charactera via the use of RegularExpressions. Then it reverses the string by pinpointing the final index in the Array, subtracting 1 from the Length property, and using a For Loop to decrement the indicies within the locally-scoped Array until it has made a record of each value in reverse order. After comparing the reversed string to the original one, the object will generate a 'message' property based on whether or not the two strings are identical.

displayMessage()

This final function was written to display the result to the user. Specifically, it incorporates the classList.remove property in order to get rid of the "invisible" class which is preventing the results section from being on display before the user has submitted their string.
